clone locally


          git clone https://github.com/YOUR_USER_NAME/appinventor-sources.git
# Clones your fork of the repository into the current directory in terminal

swap YOUR_USER_NAME for your real user name

configure remotes


            $ cd appinventor-sources
          

            $ git remote add upstream https://github.com/mit-cml/appinventor-sources.git

no notes for now: explain what a remote is?

.gitignore


            $ cp sample-.gitignore .gitignore

If you don't do this, you will get in all kinds of trouble

Checkout Dependecies


              $ git submodule update --init

If you want to make changes to Blockly, make sure you follow the instructions in the main repository README file.

Create AuthKey


              $ cd appinventor
            

              $ ant MakeAuthKey

Check out the instructions in the main repository README file if you want to make your key persistent.

build app inventor 2


            $ ant
